上手指南

五步走通

从安装，到第一本书，再到学徒的第一份周报

1

下载并打开

推荐优先选择 zip 版——解压后双击即可运行，当前稳定性更好。NSIS 安装版亦可使用。首次启动时，会让你选择一个文件夹用于保存阅读数据。

💡 首次运行提示：因为软件尚未签名，Windows 会弹出 SmartScreen 警告——点击左下角「更多信息 → 仍要运行」即可。macOS 同理：右键 → 打开 → 在弹窗中再次点击「打开」。

2

配置 API Key

在设置中填入一把 API Key 即可开始对话。推荐优先配置 智谱 GLM——国内可直连、带免费额度。获取方式见下方 API Key 获取。

3

导入第一本文献

左侧 + 将文件导入（PDF / EPUB / TXT / DOCX 均支持）。扫描版可在右键菜单中选择 OCR 识别，公式与表格将被完整保留。

4

边读边问，顺手记录

选中任意段落，右键向 AI 提问，对话将自动保存在注释中。整理长篇思路时，可打开笔记面板，将引用拖入其中汇集成稿，需要分享时再导出。

5

等学徒的观察报告

Agent 面板 → 学徒日记。学徒会记录你本周阅读了什么、在何处停下、立场是否发生变化，每周呈交一份观察。数据不足时，它不会勉强凑数。

API Key 怎么拿 · 点开看对应步骤

智谱 GLM推荐起步 · 国内 · 免费额度
  1. 去 open.bigmodel.cn 注册 + 实名
  2. 「API Keys」页生成
  3. 拾卷 → 设置 → AI Provider → GLM → 粘贴
DeepSeek国内手机号 · 便宜
  1. 去 platform.deepseek.com 注册
  2. API Keys 页生成
  3. 模型填 deepseek-chat 或 deepseek-reasoner
月之暗面 Kimi长上下文
  1. 去 platform.moonshot.cn 注册
  2. API Keys 页生成
  3. 长文献用 moonshot-v1-128k
字节豆包用 endpoint ID
  1. 去 火山方舟 控制台 → API Key
  2. 为模型创建 inference endpoint（豆包特有）
  3. 模型字段填 endpoint ID（ep-xxxxxx），不是模型名
OpenAI需要海外手机号 + 信用卡
  1. 去 platform.openai.com 注册
  2. API Keys 页生成，充 $5 激活
  3. 日常用 gpt-4o-mini 够了
Anthropic Claude同样需要海外资源
  1. 去 console.anthropic.com 注册
  2. Workbench → API Keys → Create Key，充值激活
  3. 已装过 Claude Code？拾卷可直接调本地 CLI，不用 API Key

安装提示 · 拾卷暂未做代码签名

因为还没买开发者签名证书（Windows EV ¥2000+/年 · Apple Developer $99/年），系统会把拾卷当成「未知应用」拦截。这不影响安全 —— 源码完全开源在 GitHub 可审计。绕过方法：

Windows · SmartScreen 拦截
弹出「Windows 已保护你的电脑」蓝色窗口 →
点击 「更多信息」 →
点击 「仍要运行」 即可
macOS · Gatekeeper 拦截
提示「无法验证开发者」无法打开 →
右键 拾卷 .app 点 「打开」 → 在弹窗里再点「打开」
或：系统设置 → 隐私与安全 → 允许「仍要打开」

💡 杀软误报也属同一原因。介意可下载源码自行 npm run dist 打包。
