上手指南

从安装到第一个学徒周报

5 步走通基础流程 · 6 家 LLM provider 任选一家配 Key 就能用

① 使用指南 · 五步快速上手

1

下载并启动拾卷

从 下载区 选 NSIS 安装版 / 便携版（Windows）或 mac zip 包。
首次启动会让你选数据目录（默认 ~/AppData/Roaming/shijuan）。便携版想保留数据可手动指定。

2

配置 AI Provider（至少一家）

右上角 设置 → AI Provider，填 API Key 和模型名。
推荐先配 智谱 GLM（国内访问 + 免费额度，4 RPM 已被主进程节流自动排队）。具体怎么拿 Key 见下方 ② API Key 获取指南。

3

导入第一本文献

左侧栏「+」→ 选 PDF / EPUB / TXT / DOCX / HTML / Markdown。
扫描版 PDF 可以右键「OCR 识别」转成可搜索文本（基于 GLM-OCR，公式 / 表格 / 多栏都能识别）。

4

边读边问 + 写笔记

选中任意段落 → 右键「问 AI」直接发问，回复会自动存到注释历史链。
想做长篇思考用 笔记 面板：从注释里拖引用块到笔记，用 #编号 跨文献关联，最后导出 .md / .html。

5

让 Hermes 学徒每周给你交一份观察报告

Agent 面板 → 学徒日记，AI 会按 ISO 周自动汇总你这周读了什么、停在哪、写了什么、问了什么、有没有摇摆立场。
数据不够时它会直说「这周记录太少」，不为了凑 5 条 bullet 硬编。

② API Key 获取指南 · 6 家任选其一

智谱 GLM 推荐起步 · 国内访问 · 免费额度

  1. 注册 open.bigmodel.cn，实名认证
  2. 「API Keys」页生成 Key（xxxx.xxxxxxxxxxxxxxxx 格式）
  3. 免费 tier 含 GLM-4-Flash 主对话 + embedding-2/3 嵌入 + web-search-pro 联网搜索
  4. 拾卷 → 设置 → AI Provider → GLM → 粘贴

⚠️ 免费 tier 是 4 RPM，拾卷已内置主进程节流自动排队不炸穿

DeepSeek 国内手机号 · 极便宜

  1. 注册 platform.deepseek.com
  2. API Keys 页生成（sk-... 格式）
  3. 注册送少量额度，之后约 ¥1 / 1M tokens（比 GPT-4o 便宜 100 倍）
  4. 模型推荐：deepseek-chat 通用 / deepseek-reasoner 有思维链

月之暗面 Kimi 长上下文擅长

  1. 注册 platform.moonshot.cn
  2. API Keys 页生成
  3. 模型：moonshot-v1-8k / moonshot-v1-128k（长上下文）

字节豆包 用 endpoint ID 不是模型名

  1. 注册 火山方舟 控制台 → API Key
  2. 注意：豆包用 endpoint ID 而非模型名，需要先在控制台为某个模型创建 inference endpoint
  3. 拾卷的「模型」字段填 endpoint ID（ep-xxxxxxxxxxxx）

OpenAI 需要海外手机号 + 海外信用卡

  1. 注册 platform.openai.com
  2. 创建 API Key（sk-... 格式）
  3. 充值最少 $5 启用
  4. 模型推荐：gpt-4o-mini（性价比）/ gpt-4o（质量）
  5. 嵌入模型：text-embedding-3-small 够用

Anthropic Claude 同样需要海外资源

  1. 注册 console.anthropic.com
  2. Workbench → API Keys → Create Key（sk-ant-... 格式）
  3. 充值后启用，支持 Claude Opus / Sonnet / Haiku
  4. 本地 CLI 方式：如果已装 Claude Code 在本机，拾卷可直接调本地 CLI（claude_cli provider），无需 API Key

安装提示 · 拾卷暂未做代码签名

因为还没买开发者签名证书（Windows EV ¥2000+/年 · Apple Developer $99/年），系统会把拾卷当成「未知应用」拦截。这不影响安全 —— 源码完全开源在 GitHub 可审计。绕过方法：

Windows · SmartScreen 拦截
弹出「Windows 已保护你的电脑」蓝色窗口 →
点击 「更多信息」 →
点击 「仍要运行」 即可
macOS · Gatekeeper 拦截
提示「无法验证开发者」无法打开 →
右键 拾卷 .app 点 「打开」 → 在弹窗里再点「打开」
或：系统设置 → 隐私与安全 → 允许「仍要打开」

💡 杀软误报也属同一原因。介意可下载源码自行 npm run dist 打包。
