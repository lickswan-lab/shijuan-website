A Quiet Library of Minds

召唤一个名家的思想方式，与你对话。

社区里的每一个 skill 都是一个可"召唤"的人格模板——由 Claude Code 按 persona-distill 框架从核心著作深度蒸馏而成，或由用户自己上传分享。 下载后放进 Claude Code 或拾卷，AI 便以这个人的思考方式回答你的问题。

声明：社区所有 skill 均为基于公开著作蒸馏的虚拟人格模型，不代表名家本人的真实立场 / 发言；肖像为 AI 艺术再现；回答为 AI 即时生成的模拟扮演，不可作为学术引用。
